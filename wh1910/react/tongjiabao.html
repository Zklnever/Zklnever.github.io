React项目总结(童佳宝)

01.14

项目bug

1,上传头像后,用户上传项目后头像不会被更改

2,控制台报错

Can't perform a React state update on an unmounted component. This is a no-op, but it indicates a memory leak in your application.

3,AntUi列表组件报错,没有key

项目亮点

1,利用AntUi里面的上传组件完成了头像上传

2,使用了可定制化的可复用的列表组件

3,利用local Storage和Session Storage存储token完成用户免登录

项目难点

1,利用local Storage和Session Storage存储token完成用户免登录

2,发表项目时自动增加积分,查看答案自动判断减少积分

3,AntUi列表组件需要给数据添加一个key
