赵康 react 项目总结

1.项目中遇到的bug:

(1)条件渲染的使用,由于数据请求是异步的,在点到第三层的时候就要考虑数据是否能取到,此时就可以使用条件渲染

(2)immutable.js的使用,如果需要遍历immutable对象里面的数组,需要用toJS将immutable数组转换成普通的数组

(3)dispatch如果接收函数,必须是函数的调用,即函数后面要带括号,本质上接收的还是一个对象,因为函数的返回值是一个对象

2.项目中的难点:

(1)爬取的数据中有一个表头下面的内容是对象,但有时又需要利用这个表头对象的某条属性来查询,这时候就涉及到高级查询,目前还未解决

(2)antd mobile的UI库用起来比较繁琐

(3)token那一块还是很麻烦

3.项目亮点:

(1)项目实现了短信验证登录,头像上传

(2)项目实现了搜索和评论

(3)项目使用了react-redux和mobx来进行组件之间的通信
