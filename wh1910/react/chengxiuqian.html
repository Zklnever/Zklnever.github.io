react 项目总结

3个BUG

1、antd UI库菜单组件花费了有1天的时间,菜单栏的高亮和折叠出现BUG

2、修改分类的时候,点击修改分类弹窗出来的时候不点确认的话input框里面的内容还是原来的内容

3、商品管理点击详情的时候数据带过去之后刷新页面数据丢失

3个难点

1、用户管理不同的职务权限不同

2、修改详情里面存在图片更新修改功能

3、修改详情的时候需要把商品分类填充到商品分类表格

3个亮点

1、添加分类的时候自动获取所在分类登记,在二级分类也可以添加一级分类

2、更改上下架状态

3、实时搜索可以根据名称或者描述搜索

4、修改和添加组件相同,修改时会把原数据带过去.
