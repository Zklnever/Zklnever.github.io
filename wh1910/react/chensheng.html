react 项目总结

3个BUG

1、在写点赞功能时，采用了替换图标的方法，用红爱心替换空的爱心，但矢量图标在替换后会有一段明显的移动，使用了各种定位、父元素定位、改变盒子状态等方法都无法解决

2、渲染商品列表页面时，从数据库传来的数据一直是空，但分别验证以及咨询他人时都没有问题，最终更改了下schema中的名称以及重新写了下接口后，bug消失

3、在商品列表点击商品添加进购物车时，不知道为什么又是传过去是一个完整的商品信息，有时候就只是一个只有商品ID的列表

3个难点

1、阿里云短信验证后台的实现

2、使用redux,immutable等,处理数据很混乱

3、购物车完整功能的实现

3个亮点

1、页面还原度较高, 各页面简洁其功能分化明了，整体效果较好

2、使用mobox作为数据缓存

3、商品分类,根据每个商品的分类状态去重之后得到的数组渲染到不同的选项卡,然后通过得到的分类状态来和库里面的状态进行对比来渲染不同的数据

4、登录注册页面优于原版APP,并设置了提醒,且实现了手机验证码验证.
